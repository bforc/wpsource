Step 4: Check the Plugin Version and File Type

Each plugin listed on WPSource includes basic details such as:

  • Plugin name

Always check that the version is up to date and compatible with your version of WordPress. Newer versions tend to include bug fixes, performance improvements, and support for the latest WordPress features.

Step 5: Download and Install the Plugin

To install a plugin from WPSource:

  1. Click the download link for your selected plugin.

  2. Save the ZIP file to your computer.

  3. In your WordPress dashboard, go to Plugins > Add New.

  4. Click Upload Plugin, choose the ZIP file, and click Install Now.

  5. Once installed, activate the plugin.

The process is simple and similar to uploading any premium plugin manually.

Step 6: Test and Evaluate the Plugin

Since you have access to premium functionality, use this opportunity to determine if the plugin truly meets your needs. If it does, you can eventually choose to purchase the official version for full support and updates if required.
